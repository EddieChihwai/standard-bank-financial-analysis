Title
Standard Bank Group Financial Performance Analysis (2014–2024)

Project Overview
This project analyzes the financial performance of Standard Bank Group over the period 2014–2024 using SQL and financial data analysis techniques. The objective is to identify the key drivers of profitability, operational efficiency, and balance sheet growth using publicly available financial statement data.

The analysis focuses on understanding how revenue growth, lending expansion, cost efficiency, and credit risk influence the bank’s return on equity over time.

Tools Used
PostgreSQL — data storage and analysis
SQL — financial data analysis and statistical queries
Power BI — data visualization and dashboarding
GitHub — project documentation and version control

Dataset
The dataset used in this project was compiled from publicly available Standard Bank financial statements.

Variables included in the dataset:

year
total_income
net_interest_income
headline_earnings
roe
nim
cost_to_income
credit_loss_ratio
gross_loans

Project Structure

Key Analytical Techniques

Year-over-year growth analysis using SQL window functions
Correlation analysis to measure relationships between financial metrics
Regression trend analysis using PostgreSQL statistical functions
Structural efficiency comparisons across time periods

Key Findings

Profitability volatility is primarily driven by credit loss cycles.
There is a strong inverse relationship between return on equity and credit loss ratios.
Revenue growth has been structurally positive over the period analyzed.
Cost efficiency improved after 2020, suggesting structural operational improvements.
Balance sheet expansion has been a major driver of growth, with loan growth outpacing revenue efficiency.
